# 怪物技能清单（Monster Skills）

此处记录怪物技能设定，用于 PVE 敌人差异化。字段与格式遵循《技能设计规范与模板》。

表格视图（概览）：

| ID | 名称 | 类型 | 触发 | 目标 | 核心参数 | 冷却 | 描述 | 表现 |
|---|---|---|---|---|---|---:|---|---|
| skill.monster.poison_bite.v1 | 毒牙 | trigger | on_attack_after_damage | enemy | dot=3x(10+0.1Atk) | 0 | 造成伤害后，附加3回合中毒（每回合10+0.1Atk）。 | fx:fx_poison |
| skill.monster.howl.v1 | 群狼嚎叫 | trigger | on_turn_start | team_allies | atk+25% dur=2 | 3 | 每3回合咆哮一次，使所有友方攻击+25%，持续2回合。 | fx:fx_howl sfx:sfx_howl |
| skill.monster.rend_all.v1 | 群体撕咬 | active | on_attack_before_damage | team_enemies | splash=40% | 2 | 攻击前对所有敌方造成40%溅射伤害。 | fx:fx_bite |
| skill.monster.self_explode.v1 | 自爆 | active | on_death | team_enemies | rate=1.2 | 0 | 死亡时对所有敌人造成1.2倍伤害。 | fx:fx_explode |
| skill.monster.regeneration.v1 | 快速再生 | passive | on_turn_end | self | heal=5%maxhp | 0 | 每回合结束恢复最大生命5%。 | fx:fx_heal_green |

规范化定义（示例）：

```yaml
# 毒牙
id: skill.monster.poison_bite.v1
name: 毒牙
type: trigger
scope: monster
triggers:
  - event: on_attack_after_damage
    targets: enemy
    effects:
      - type: dot
        stat: hp
        flat: 10
        rate: 0.10    # 0.1 * 攻击
        duration: 3
visual:
  fx_id: fx_poison
desc: 造成伤害后，附加3回合中毒（每回合10+0.1Atk）。
tags: ["poison","dot"]
```

```yaml
# 群狼嚎叫
id: skill.monster.howl.v1
name: 群狼嚎叫
type: trigger
scope: monster
cooldown: 3
triggers:
  - event: on_turn_start
    targets: team_allies
    conditions:
      - round_mod: { mod: 3, eq: 0 }
    effects:
      - type: buff
        stat: attack
        rate: 0.25
        duration: 2
visual:
  fx_id: fx_howl
  sfx_id: sfx_howl
desc: 每3回合咆哮一次，使所有友方攻击+25%，持续2回合。
tags: ["support","aoe"]
```

```yaml
# 自爆
id: skill.monster.self_explode.v1
name: 自爆
type: active
scope: monster
triggers:
  - event: on_death
    targets: team_enemies
    effects:
      - type: damage
        rate: 1.2
visual:
  fx_id: fx_explode
desc: 死亡时对所有敌人造成1.2倍伤害。
tags: ["burst","death"]
```

> 备注：怪物技能可附带“AI权重/优先级”字段，以指导怪物在多个可用技能中选择释放顺序；也可在条件中加入 `hp_pct_below`、`ally_count_ge` 等表达式，提升敌人行为的可读性与可调性。
