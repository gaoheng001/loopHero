## 标准化定义（YAML，用于落地实现）

### 队员档案（1001：紫菱）
```yaml
id: hero.1001
name: 紫菱
base_stats:
  attack: 100
  defense: 80
  hp: 500
skills:
  - skill.hero.wanjian_guizong.v1
  - skill.hero.ningxin_jue.v1
```

### 主动技能：万剑归宗
```yaml
id: skill.hero.wanjian_guizong.v1
name: 万剑归宗
type: active
scope: hero
cooldown: 0
triggers:
  - event: on_turn_start
    targets: team_enemies
    conditions:
      - chance: 0.30                 # 技能释放几率：30%
      - first_round_chance_multiplier: 2.0  # 首回合几率*2
      - max_uses_per_battle: 1       # 释放最大次数：1
    effects:
      - type: damage
        rate: 0.50                   # 技能伤害率：50%
        damage_type: physical        # 物理伤害
        formula: "damage = enemy_team_count * rate"  # 伤害计算：敌方小队成员数 * 伤害率
        targets: team_enemies
visual:
  fx_id: fx_sword_rain
  sfx_id: sfx_sword_rain
desc: 对敌方小队整体发动剑雨，每名敌人承受等于50%系数的伤害，总伤害随敌方人数线性叠加（仅本场战斗最多释放1次，首回合触发概率加倍）。
tags: ["aoe","physical","burst"]
```

### 被动技能：凝心决
```yaml
id: skill.hero.ningxin_jue.v1
name: 凝心决
type: passive
scope: hero
triggers:
  - event: on_battle_start
    targets: self
    effects:
      - type: buff
        stat: skill_crit_rate
        rate: 0.20     # 自身释放技能暴击率+20%
        duration: 0    # 0表示整场战斗持续
  - event: on_battle_start
    targets: team_allies
    effects:
      - type: buff
        stat: normal_attack_crit_rate
        rate: 0.10     # 小队整体普攻暴击率+10%
        duration: 0
visual:
  fx_id: fx_focus
desc: 稳心静气，自身技能暴击率提高20%，同时为我方全队提供普攻暴击率+10%的持续增益。
tags: ["crit","aura","team"]
```
